# Moodboard Summary — Institut La Clé

---

## UNIVERS VISUEL PERÇU

### En une phrase
Un institut qui parle comme une bibliothèque de nuit : tout est sombre, calme, structuré, et la seule lumière vient du bronze — comme une lampe de lecture dans l'obscurité.

### Références visuelles proches
- **Aesop** (cosmétiques) — minimalisme radical, typographie serif, tons sombres
- **Kinfolk Magazine** — sobriété éditoriale, espaces vides, lenteur assumée
- **Maison d'édition Gallimard** — prestige discret, sérif, institution
- **Musée Louvre Abu Dhabi** — architecture de lumière filtrée dans l'obscurité
- **The Row** (Olsen) — luxe silencieux, noir dominant, sans logo visible
- **Terminal de Bloomberg** — fond noir, typographie fonctionnelle, données épurées

### Ce qui différencie La Clé des autres sites de formation
| Marché standard | La Clé |
|----------------|--------|
| Fond blanc/clair | Fond noir radical |
| Couleurs vives, dégradés multicolores | Monochrome + bronze unique |
| Photos sourire/équipe/salle | Zéro photographie |
| Coins arrondis, formes douces | Angles droits stricts |
| Animations bounce/spring | Animations lentes, eased |
| CTA agressifs "INSCRIVEZ-VOUS" | CTA sobres "En savoir plus" |
| Témoignages, preuves sociales | Contenu purement conceptuel |
| Navigation riche | Navigation séquentielle, pas de menu |
| Font sans-serif partout | Serif pour les titres (Cormorant) |
| Fond statique | Aurora atmosphérique vivante |

## DIRECTIONS CRÉATIVES EXPLOITABLES

### 1. "La Clé dans le réel"
Décliner l'univers digital dans des supports physiques :
- Papeterie noire, embossage bronze
- Certificats sur papier sombre, typographie dorée
- Packaging sobre pour supports de cours

### 2. "La Clé en mouvement"
Exploiter les animations pour du motion design :
- Intro vidéo : stroke draw du symbole → baseline → aurora reveal
- Transitions entre slides : fadeInUp institutional
- Fonds animés pour stories/reels : aurora en boucle

### 3. "La Clé éditoriale"
Créer du contenu Instagram comme un magazine :
- Une pensée par jour, design minimaliste
- Carrousels "modules" réutilisant le design timeline
- Série de cartes-concepts avec numérotation décorative

### 4. "La Clé atmosphérique"
Jouer sur l'atmosphère unique :
- Vidéos d'ambiance (aurora + typographie qui apparaît)
- ASMR visuel : animations lentes, transitions douces
- Captures d'écran stylisées du site comme contenu

## ASSETS CLÉS POUR L'ÉQUIPE CRÉA

### Pour le directeur de communication
1. `logos/` — Toutes les variantes du logo et du symbole
2. `colors/palette.css` — Copier-coller direct des couleurs
3. `colors/palette.svg` — Planche visuelle pour briefs
4. `exports/social-pack/` — Templates prêts à l'emploi
5. `animations/splash-screen.html` — L'animation signature à montrer

### Pour un motion designer
1. `animations/` — Tous les patterns d'animation avec timing
2. `backgrounds/aurora-atmosphere.html` — Aurora en boucle, capturable en vidéo
3. `design-system-summary.md` — Easings customs et durées

### Pour un graphiste print
1. `logos/key-symbol.svg` et variantes — Vectoriel natif
2. `typography/typography-specimen.html` — Tous les styles
3. `colors/palette.svg` — Référence couleurs
4. Polices : Google Fonts (Cormorant Garamond + DM Sans) — licence OFL, libre d'utilisation
