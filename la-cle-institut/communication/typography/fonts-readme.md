# Fonts — Institut La Clé

## Polices utilisées

### Cormorant Garamond
- **Source** : Google Fonts
- **Licence** : SIL Open Font License (OFL) — libre pour tout usage commercial
- **URL** : https://fonts.google.com/specimen/Cormorant+Garamond
- **Poids utilisés** : 300 (Light), 400 (Regular), 500 (Medium), 600 (SemiBold), 700 (Bold)
- **Styles** : Normal + Italic
- **Usage** : Titres (H1–H6), logo texte, chiffres décoratifs, baseline
- **Variable CSS** : `--font-cormorant`

### DM Sans
- **Source** : Google Fonts
- **Licence** : SIL Open Font License (OFL) — libre pour tout usage commercial
- **URL** : https://fonts.google.com/specimen/DM+Sans
- **Poids utilisés** : 300 (Light), 400 (Regular), 500 (Medium), 700 (Bold)
- **Usage** : Corps de texte, labels, boutons, navigation, tags
- **Variable CSS** : `--font-dm-sans`

## Fichiers WOFF2

Les 12 fichiers `.woff2` dans ce dossier sont les subsets Latin des deux polices,
auto-générés par Next.js via `next/font/google`. Ils couvrent tous les poids utilisés.

## Installation pour usage hors-web

1. Télécharger depuis Google Fonts (liens ci-dessus)
2. Installer les fichiers TTF/OTF sur votre système
3. Dans les logiciels Adobe/Figma, les polices apparaîtront directement
